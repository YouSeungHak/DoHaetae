# 도해태 (DoHaetae)

> macOS 바탕화면 위에 살아있는 AI 해태 캐릭터 — 당신이 막혔을 때 먼저 말을 건다.

## 한 줄 소개

AI 수호 생명체 해태가 바탕화면에 상주하며 사용자의 작업을 관찰하고, 막힌 것 같을 때 조용히 도움을 제안합니다.

## 기술 스택

| 항목 | 선택 |
| --- | --- |
| 플랫폼 | macOS |
| 프레임워크 | Electron |
| 언어 | TypeScript |
| AI | Claude API (Anthropic) |

## 프로젝트 구조

```
003_DoHaetae/
├── docs/

│   ├── 03_design/       설계 문서
│   ├── 04_ux_ui/        UX/UI 문서
│   └── 05_qa/           QA 문서
├── reports/             작업 보고서

```

## 핵심 컨셉

- 해태 캐릭터 3종 (차분 / 장난꾸러기 / 똑똑이) 중 랜덤 선택
- 바탕화면 투명 오버레이 + 항상 위
- 키보드 무입력 감지 → 말풍선 표시
- AI (Claude) 연동으로 컨텍스트 기반 도움 제공
